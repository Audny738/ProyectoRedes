Seguridad en la web



La web

Conforme la tecnología avanza cada día se nos ofrecen muchas facilidades para cada una de nuestras necesidades. Un lugar donde podemos acudir para ello es la Web. Si bien sus inicios no se pueden comparar con el alcance y poder que tiene ahora que prácticamente cuenta con una alta gama de herramientas y presentaciones. Un ejemplo de ello es la primera página web, aparecida el 23 de agosto de 1991, la cual no contaba con imágenes, videos u otros recursos con los que cuentan los sitios web en la actualidad. Así resumimos brevemente la Historia de las Páginas Web.

Web Security vs Website Security

Debido a la similitud que presentan en su escritura normalmente se llega a confundir ambos conceptos pero aquí se aclara esta confusión.

Primero la seguridad web es sinónimo de ciberseguridad y es una sombrilla para la seguridad de sitios web pues también implica seguridad de la nube y la seguridad de aplicaciones web que consisten en la protección de la nube y las aplicaciones web, respectivamente. La protección de un VPN también cae dentro de la seguridad web.

Hoy en día se estiman que existen alrededor de 1.700 millones de sitios web y, aunque los recursos y oportunidades aumentan, también lo hacen los peligros como los crackers que intentan poseer información sensible y amenazando nuestra seguridad. Por ello se llega a un concepto que es igual de importante a la hora de crear una página web como lo es el Frontend y Backend, ese concepto es la Seguridad de la Web.
