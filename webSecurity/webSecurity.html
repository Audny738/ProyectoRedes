La seguridad en la Web

Conforme la tecnología avanza cada día se nos ofrecen muchas facilidades para cada una de nuestras necesidades. Un lugar donde podemos acudir para ello es la Web. Si bien la Web a sus inicios no es para nada comparable con lo que se tiene ahora. Por ejemplo, la primera página web, aparecida el 23 de agosto de 1991, no contaba con imágenes, videos u otros recursos con los que cuentan los sitios web en la actualidad. Así resumimos brevemente la Historia de las Páginas Web.

Al día de hoy se estiman que existen alrededor de 1.700 millones de sitios web y, aunque los recursos y oportunidades aumentan, también lo hacen los peligros donde cada vez el número de crackers aumenta e intenta poseer información sensible y amenazando nuestra seguridad. Por ello se llega a un concepto que es igual de importante a la hora de crear una página web como lo es el Frontend y Backend, ese concepto es la Seguridad de la Web.

¿Qué es la Seguridad de la Web?

La Seguridad de la Web es cualquien acción llevada a cabo para asegurar que los datos no sean expuestos a ciberciminales o para prevenir la explotación del sitio de alguna forma o de los datos que se transporta por el internet a través de la red. Asegurarse que nuestros sitios o aplicaciones web tengan seguridad es algo sumamente crítico porque hoy en día existen incontables peligros, con el cual hasta el simple bug puede resultar en la filtración de información sensible.
